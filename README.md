# Sean Esla Portfolio Website

A personal portfolio website showcasing Sean's experiences and skills in Math, Computer Science, aviation, and internships at NASA NCAS and Caltech's AT3 program.

## Features

- **Interactive Hero Section**: Features a dynamic typing animation displaying different roles/experiences
- **About Section**: Personal bio with image gallery
- **Experience Section**: 
  - NASA NCAS Missions 1 & 2 with embedded research poster
  - Caltech AT3 program description
- **Skills Section**: Animated skill bars for different programming languages
- **Contact Section**: Links to LinkedIn and downloadable resume

## Tech Stack

This project uses vanilla web technologies:
- HTML5
- CSS3 
- JavaScript (no frameworks)

## File Structure

- `index.html` - Main content structure
- `style.css` - All styling information
- `script.js` - Interactive elements and animations
- `imagesandpdfs/` - Images and PDF documents
- `logos/` - Organization logos used in the experience section

## Development Notes

The site features:
- Southern California inspired color palette
- Responsive design
- Custom typing animation
- PDF embedding
- Light grid background pattern

## Running Locally

To run the website locally, simply open `index.html` in your web browser or use a local development server:

```bash
# Using Python's built-in server
python -m http.server 8080

# Open in browser
http://localhost:8080
```
